## YummySphere Resturant and Food Delivery Website

A responsive food delivery website built with HTML, CSS, and Bootstrap 5.

## Features

- Fully responsive design (mobile, tablet, desktop)
- Interactive navigation with Bootstrap scrollspy
- Dish category filtering using Bootstrap tabs
- Modal popups for ordering and cart
- Collapsible "See All" section
- Accordion for additional story details
- Tooltips on interactive elements
- Bootstrap components for all functionality
